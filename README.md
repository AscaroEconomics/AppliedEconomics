# Economia applicata in Python

Questa repository raccoglie le analisi effettuate per la realizzazione degli articoli pubblicati sul mio profilo Substack dedicato all’economia applicata.

In particolare, all'interno della sotto-cartella <b>notebooks</b> sono visionabili i Jupiter Notebook con il codice utilizzato per le varie analisi e, quando necessario, nella cartella <b>data</b> è possibile accedere ai dati utilizzati nelle stesse.

L’obiettivo è esplorare se e come i modelli teorici trovino applicazione
nei dati reali.

## Contenuti
- Regressione Lineare e OLS: un'applicazione pratica.
